# Voting System (C)

## Description
This project is a simple voting system developed using C language.
It allows users to cast votes for candidates, stores votes using file handling, and displays the final results.

## Features
- Cast vote for candidates
- Store votes in file
- Count votes
- Display results and winner

## How to Run
1. Compile the program using any C compiler
2. Run the program
3. Choose options from the menu
